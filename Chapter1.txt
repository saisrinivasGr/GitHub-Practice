

Is CVS is configuration management tool?
What Does Concurrent Versions System (CVS) Mean? Concurrent versions system (CVS) is an
 open-source software configuration management utility designed to manage different versions
 of the same software project in a specialized repository.

What is a CVS module?
CVS modules are just symbolic names for sets of files, generally just symbolic names for a
particular directory in the repository. example, the module "news-template" refers to
the path leland/news/template in the repository.
